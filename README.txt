SPINDALE ZONING DASHBOARD

Open index.html to launch the connected dashboard and parcel map.

How it works:
1. Search for an address or parcel number, or click a parcel on the map.
2. The parcel address, number, and zoning district appear in the dashboard.
3. Multi-zoned parcels show every district as a selectable button.
4. Parcel 421795 includes its conditional-zoning terms and signed agreement.
5. Search for a proposed use in the dashboard to see its zoning status.
6. Changing the dashboard district moves the map to that zoning district.
7. Clear Search clears both the dashboard search and the selected map parcel.

Keep this entire folder together. The map depends on the included data, CSS,
JavaScript, legend, font, and image folders.

The original dashboard and ZIP files were not changed.
